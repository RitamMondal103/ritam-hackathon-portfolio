# 🌍 Make Your Website Accessible Anywhere, Anytime!

## 🚀 **Quick Deployment Options (Choose One)**

### **Option 1: GitHub Pages (FREE - Recommended)**
**Perfect for: Personal portfolios, static websites**

#### **Step 1: Create GitHub Repository**
```bash
# Open Command Prompt/PowerShell
cd c:\Users\User\CascadeProjects\personal-website
git init
git add .
git commit -m "Initial commit - Ritam's Hackathon Portfolio"
git branch -M main
```

#### **Step 2: Create GitHub Repository**
1. Go to https://github.com/new
2. Repository name: `ritam-hackathon-portfolio`
3. Description: "Personal hackathon portfolio with collaboration features"
4. Make it **Public**
5. Click "Create repository"

#### **Step 3: Push to GitHub**
```bash
git remote add origin https://github.com/YOUR_USERNAME/ritam-hackathon-portfolio.git
git push -u origin main
```

#### **Step 4: Enable GitHub Pages**
1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** → **/ (root)**
5. Click **Save**

#### **Your Website Will Be Live At:**
```
https://YOUR_USERNAME.github.io/ritam-hackathon-portfolio
```

### **Option 2: Netlify (FREE - Custom Domain Ready)**
**Perfect for: Custom domains, advanced features**

#### **Step 1: Push to GitHub** (Same as Option 1)

#### **Step 2: Deploy to Netlify**
1. Go to https://netlify.com
2. **Sign up** with GitHub
3. Click **"New site from Git"**
4. Choose your `ritam-hackathon-portfolio` repository
5. Build settings:
   - Build command: `echo "No build needed"`
   - Publish directory: `.`
6. Click **"Deploy site"**

#### **Your Website Will Be Live At:**
```
https://random-name-123456.netlify.app
```

### **Option 3: Vercel (FREE - Fastest)**
**Perfect for: Performance, global CDN**
